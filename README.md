# goose-game

A Java 11 exercise. Implement a console goose-game.


## Usage

Requirements:

- **Java JDK 11**
- Maven 3.6.0

Development environment:

- Visual Studio Code
- OpenJDK 11

Run unit tests:

    mvn test

Run console application:

    mvn compile
    $JAVA_HOME/bin/java -cp ./target/classes icardi.goose.game.App
